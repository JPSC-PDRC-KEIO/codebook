.. rst-class:: answer
選択肢 / 記入欄
======================

  
     1. 全ての商品・サービスで同じ税率がよい
  
     2. 生活に欠かせない飲食料品だけは、税率を下げる方がよい
  
     3. 酒や外食も含めた飲食料品全般は、税率を下げる方がよい
  
     4. 飲食料品に限らず商品・サービスの内容や特性に合せて、税率に差をつける方がよい
