.. rst-class:: question

質問文
==================


   あなた方ご夫婦は、今住んでいる家、セカンドハウス以外に、例えば他人に貸すために土地・ 建物などの不動産を所有していますか。（親との共同名義のものも含みます）



.. rst-class:: answer

選択肢 / 記入欄
======================

  1. 土地のみを所有している
  2. 建物のみを所有している
  3. 土地、建物両方を所有している
  4. マンションを所有している
  5. 所有していない
