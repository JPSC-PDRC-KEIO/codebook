========================================
調査対象者の番号（ID）
========================================


変数名：ID
-----------------


説明
-----------------

調査対象者に一意に割り振った数字です。
この数字によって、その対象者が調査に参加した調査回と出生コーホート（ :doc:`ref </id_panel/cohort>` )も区別することができます。
